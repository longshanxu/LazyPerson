# V3.6 线位浅黄色与单方向趋势通道调整方案

## 背景

V3.5 将普通线位统一为绿色，但在深色行情图上仍显得偏花。趋势线同时展示向上和向下两组，也会让主图信息过载。

## 实现范围

### 线位颜色

- 保留重点线规则：
  - `+20%` 红色。
  - `+50%` 蓝色。
  - `+80%` 白色。
- 其他百分比线统一改为浅黄色。
- 浅黄色和白色标签使用深色文字，提高可读性。
- 颜色缓存版本升级到 `v3`，避免旧绿色缓存影响新默认展示。

### 趋势通道

- 近 90 自然日交易窗口拆成前后两个时间段。
- 分别识别两个时间段内的高点和低点。
- 用前段低点连接后段低点，形成通道下轨。
- 用前段高点连接后段高点，形成通道上轨。
- 只保留当前方向的一组通道：
  - 后段价格中枢高于前段，显示向上通道。
  - 后段价格中枢低于前段，显示向下通道。
- 向上通道使用红色。
- 向下通道使用绿色。

## 验收标准

- 普通百分比线为浅黄色。
- `+20%`、`+50%`、`+80%` 继续分别为红、蓝、白。
- 图上只显示一组趋势通道，共 2 条线。
- 向上通道为红色，向下通道为绿色。
- `npm run test` 和 `npm run build` 通过。
