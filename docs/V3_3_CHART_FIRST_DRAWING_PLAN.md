# V3.3 走势图优先与重点画线调整方案

## 背景

V3.2 已切换为深色行情终端风格，但走势图占用面积仍不足。当前自选股和个股信息默认常驻左右两侧，会压缩 K 线主图。自动画线的线位标签也需要贴近用户截图风格，重点线位需要更醒目。

## 调整目标

1. 页面默认以走势图为第一主体。
2. 自选股列表不默认占位，改为按钮打开详情列表。
3. 个股基本信息不默认占位，改为按钮打开详情面板。
4. 指标区只保留：
   - MACD。
   - LON。
5. 前后端指标请求只保留 `macd,lon`。
6. 自动画线全部使用实线。
7. 趋势线全部使用实线，确保上涨趋势和下跌趋势都可见。
8. 水平线标签参考截图，改为左侧对齐的彩色标签。
9. `+20%`、`+50%`、`+80%` 三条线作为重点线：
   - `+20%` 红色。
   - `+50%` 黑色。
   - `+80%` 蓝色。
   - 线条加粗。

## LON 指标规则

采用通达信常见 LON 长线趋势公式：

```text
LC = REF(CLOSE, 1)
VID = SUM(VOL, 2) / ((HHV(HIGH, 2) - LLV(LOW, 2)) * 100)
RC = (CLOSE - LC) * VID
LONG = SUM(RC, 0)
DIFF = SMA(LONG, 10, 1)
DEA = SMA(LONG, 20, 1)
LON = DIFF - DEA
LONMA = MA(LON, 10)
```

## 验收标准

- 默认页面主图明显变大。
- 自选股通过按钮打开，不默认占用左侧空间。
- 个股信息通过按钮打开，不默认占用右侧空间。
- 指标 tab 只显示 `MACD` 和 `LON`。
- 接口指标参数为 `macd,lon`。
- 图上所有自动水平线为实线。
- `+20%`、`+50%`、`+80%` 加粗，并分别显示红、黑、蓝。
- 上涨趋势线和下跌趋势线均为实线。
- 水平线标签位于图表左侧。
- `npm run test` 通过。
- `npm run build` 通过。
