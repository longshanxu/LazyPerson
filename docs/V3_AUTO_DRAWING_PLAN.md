# V3 自动画线方案

## 背景

V2 已经具备 A 股工作台、K 线图、成交量、指标 tab、自选股和数据状态提示。V3 开始增加高级定制分析能力。

本阶段先实现“自动画线”：用户不手动画线，系统按最新日 K 的近期高低点自动生成价格线，并标记趋势方向。

测试标准标的：`002138`。

## 目标

- 基于最新日 K 自动识别近期最高点和近期最低点。
- 自动判断近期趋势是向上还是向下。
- 自动绘制 5 条价格线。
- 每条价格线之间按 10% 幅度间隔。
- 页面上展示趋势方向、基准点、近期高低点和线位说明。

## 计算范围

默认使用最近 120 根日 K。

原因：

- 足够覆盖一个中短期波段。
- 不会被过远历史高低点干扰。
- 对日常看盘响应较快。

后续可以升级为可配置：

- 60 日。
- 120 日。
- 250 日。
- 自定义窗口。

## 趋势判断

在最近 120 根日 K 中：

1. 找到最高价对应的 K 线，记为近期最高点。
2. 找到最低价对应的 K 线，记为近期最低点。
3. 如果近期最高点出现时间晚于近期最低点，判断为向上趋势。
4. 如果近期最低点出现时间晚于近期最高点，判断为向下趋势。

```text
低点 -> 高点：向上趋势
高点 -> 低点：向下趋势
```

## 画线规则

### 向上趋势

以近期最低点为基准，从低点向上画 5 条线：

```text
第 1 条：低点 * 1.10
第 2 条：低点 * 1.20
第 3 条：低点 * 1.30
第 4 条：低点 * 1.40
第 5 条：低点 * 1.50
```

含义：

- 作为上涨目标位或压力观察位。
- 每条线之间相对基准低点间隔 10%。

### 向下趋势

以近期最高点为基准，从高点向下画 5 条线：

```text
第 1 条：高点 * 0.90
第 2 条：高点 * 0.80
第 3 条：高点 * 0.70
第 4 条：高点 * 0.60
第 5 条：高点 * 0.50
```

含义：

- 作为下跌支撑位或风险观察位。
- 每条线之间相对基准高点间隔 10%。

## 图表表现

K 线图上显示：

- 5 条自动价格线。
- 向上趋势使用红色系。
- 向下趋势使用绿色系。
- 每条线显示标签：
  - `+10%`
  - `+20%`
  - `+30%`
  - `+40%`
  - `+50%`
  - 或向下趋势的 `-10%` 到 `-50%`

工作台中显示：

- 趋势方向：向上 / 向下。
- 近期最高点：日期 + 价格。
- 近期最低点：日期 + 价格。
- 当前收盘价。
- 距离最近线位的差值。

## V3 本阶段不做

- 不做手动画线。
- 不做保存自定义线。
- 不做多套画线模板。
- 不做突破提醒。
- 不做回测。

这些可以作为 V3 后续增强。

## 验收标准

- `002138` 可作为默认测试标的出现在自选股。
- 打开 `002138` 日 K 后，图表自动出现 5 条线。
- 页面显示趋势方向。
- 页面显示近期最高点和近期最低点。
- 5 条线的间隔符合 10% 规则。
- 非日 K 周期不强制显示日 K 自动线，避免误导。
- `npm run test` 通过。
- `npm --prefix frontend run build` 通过。
- 浏览器验证页面正常显示。
