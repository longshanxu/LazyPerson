# MVP 验收清单

## 后端

- [ ] `GET /api/health` 可用。
- [ ] `GET /api/symbols/search?q=600519` 可返回股票信息。
- [ ] `GET /api/quotes/realtime?symbols=600519` 可返回实时行情。
- [ ] `GET /api/kline/600519?period=day&indicators=ma,macd,rsi` 可返回日 K 和指标。
- [ ] `GET /api/kline/600519?period=5m` 可返回分钟 K。
- [ ] `GET /api/money-flow/600519` 可返回资金流或明确的不可用提示。
- [ ] 所有行情响应包含 `quality.source`、`quality.from_cache`、`quality.updated_at`。
- [ ] 数据源失败时，已缓存数据可作为兜底返回。

## 缓存

- [ ] SQLite 数据库自动创建。
- [ ] K 线 Parquet 文件自动创建。
- [ ] 日 K 支持增量更新。
- [ ] 实时行情和分钟线有 TTL。
- [ ] 手动刷新参数 `refresh=true` 生效。
- [ ] 可清理指定股票或指定类型缓存。

## 前端

- [ ] 首页展示自选股看板。
- [ ] 支持添加、删除自选股。
- [ ] 看板展示最新价、涨跌幅、成交额、换手率、更新时间。
- [ ] 股票详情页展示 K 线和成交量。
- [ ] 支持日 K、5 分钟、15 分钟、30 分钟、60 分钟切换。
- [ ] 支持 MA、MACD、RSI 显示。
- [ ] 支持资金流面板。
- [ ] 数据加载、失败、缓存兜底状态清晰。

## 质量

- [ ] 单只股票数据标准化有单元测试。
- [ ] 指标计算有单元测试。
- [ ] 缓存读写有单元测试。
- [ ] 前端关键页面无明显布局溢出。
- [ ] README 有启动、配置和免责声明。

## 第一版不做

- [ ] 不做交易下单。
- [ ] 不做荐股。
- [ ] 不做云端同步。
- [ ] 不做多用户权限。
- [ ] 不承诺实时数据的交易级准确性。
